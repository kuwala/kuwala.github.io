
# Advanced Pico8
The chiptunes of Game Dev.

* Menuitems
* Coroutines

## Coroutines
cocreate(f) - creates a coroutine of function f and returns Coroutines
coresume(c) - resumes coroutine c
costatus(c) - returns "dead", "running", "suspended"
